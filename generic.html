Semana 1: Introducción al Desarrollo de Páginas Web

En la primera semana, aprendimos a desarrollar una página web utilizando Visual Studio Code con HTML. Durante esta sesión, se introdujeron los conceptos básicos del desarrollo web, incluyendo la estructura de un documento HTML, cómo se organizan los elementos en una página, y la importancia de un entorno de desarrollo adecuado. Aprendí a:

Crear un documento HTML básico. Entender la estructura de etiquetas como html, head, body, y su propósito. Integrar contenido básico, como texto y encabezados, en una página web. Guardar y visualizar mi página en un navegador.

Visual Studio Code

Visual Studio Code (VS Code) es un editor de código fuente ligero y potente desarrollado por Microsoft. Es compatible con una amplia gama de lenguajes de programación y ofrece características avanzadas como autocompletado, depuración integrada, y control de versiones con Git. VS Code es altamente personalizable mediante extensiones y es una herramienta multiplataforma, disponible para Windows, macOS y Linux.

Durante la primera semana del curso, descubrí la extensión Live Server en Visual Studio Code, y rápidamente se convirtió en una herramienta indispensable para mi flujo de trabajo. Antes de usar Live Server, cada vez que hacía un cambio en mi código HTML o CSS, tenía que guardar el archivo, cambiar al navegador y actualizar la página manualmente para ver los resultados. Este proceso era repetitivo y consumía mucho tiempo.

Live Server simplificó todo este proceso. Al activar la extensión, se crea un servidor local que me permite visualizar mi página web en tiempo real en el navegador. Ahora, cada vez que guardo un archivo, la página se actualiza automáticamente, mostrando los cambios de inmediato. Esta funcionalidad ha sido increíblemente útil, ya que me permite ver al instante cómo mis modificaciones afectan el diseño y la funcionalidad de la página.

En resumen, Visual Studio Code ha sido una herramienta clave que me ha permitido trabajar de manera más eficiente, organizada y con mayor flexibilidad en mis proyectos de desarrollo.
